#TitleBarView
--------------------------
一、简介：

一个支持沉浸式 TitleBarView，支持文xml及java代码设置各种属性

**1.1 Gradle集成**

```
allprojects {
    repositories {
        ...
        maven { url "https://jitpack.io" }
    }
}
```

```
dependencies {
     compile 'com.github.AriesHoo:TitleBarView:1.2.0'
}
```
